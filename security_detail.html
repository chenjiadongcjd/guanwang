深圳欢乐海岸安防平台

SHENZHEN HAPPY COAST SECURITY PLATFORM

深圳欢乐海岸安防项目是火元素为深圳国际创意生活空间-欢乐海岸开发的集应急预案、预案部署、警力检查、现场指挥为一体的综合性安防平台，利用虚拟场景漫游及增强显示等VR前沿技术，实现了欢乐海岸最为真实的数字场景，并结合现场监视系统，提供了多种视图场景，为欢乐海岸的运营安全保驾护航。

预案管理

PLAN MANAGEMENT

预言管理模块实现了预案的组织架构、预案等级、预案地点标记等信息的管理，安防管理人员可为各种事件和活动组织预案，提前落实事件和活动中的安保责任人。

预案部署

PRE-DEPLOYMENT

预案部署模块将安保指挥、医疗救助、便衣侦查、后勤保障、反恐处突等各种安防资源类型融入系统。
结合欢乐海岸3D数字实景位置坐标，为预案部署有明确空间位置的安防资源，大大提高了安防管理人员对安防态势的掌握及安防人员对责任区域指派的认知。

警力检查

POLICE CHECK

警力检查模块现实了安保指挥、联络调度、应急备勤、秩序管理、流动巡查等信息的管理，将欢乐海岸安防中的警力位置部署、巡逻路径等信息通过3D实景、地图、地标技术直观的展示给管理人员，帮助管理人员对安防时态掌握。

现场指挥

ON-SITE COMMAND

安保负责人上报事件发生，管理人员可以通过现场指挥模块快速定位发生事件位置，以及快速调动周边安保资源前往事件地点协助处理。
现场指挥模块中还能根据安保人员位置找到最近监视摄像头查看现场，能够一目了然的了解事件以及准确高效的处理事件。
